##文档
-------

app 是每个项目不一样的 可以自定义的

/ui 是核心文件 里面的组件要二开的话 可以复制到/app 下进行二开

以后只会更新/ui 下的文件 你可以直接复制 /ui 来更新3.2 以后的版本

当前版本为Beta版本，还未完全全部功能，欢迎大家提交PR，好的代码会考虑合并的哦！
